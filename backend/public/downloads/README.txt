APK FAYLINI SHU YERGA QO'YING
==============================

Android Studio'da APK yig'gandan keyin, faylni shu papkaga nusxalang va
nomini aniq "dls-bot.apk" qiling:

    backend/public/downloads/dls-bot.apk

Shundan keyin sayt (http://SERVER:3000/) dagi "APK yuklab olish" tugmasi ishlaydi.

APK yig'ish (Android Studio):
  Build > Build Bundle(s) / APK(s) > Build APK(s)
  Tayyor fayl: android/app/build/outputs/apk/debug/app-debug.apk
  Uni shu papkaga "dls-bot.apk" nomi bilan ko'chiring.
